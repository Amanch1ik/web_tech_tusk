history_edu Этапы развития веб-технологий

  1. 1

    Web 1.0 — Эра чтения

    Статические HTML-страницы, гостевые книги и начало цифровой эры. Контент создавался немногими для многих.

  2. 2

    Web 2.0 — Эра участия

    Социальные сети, облачные вычисления и динамический контент. Пользователи стали создателями контента.

  3. 3

    Web 3.0 — Эра владения

    Блокчейн, децентрализованные протоколы и искусственный интеллект. Интернет с упором на приватность и право собственности.
