# Самостоятельная работа — Web-технологии

**Тема:** Современные веб-технологии (Вариант 20)

## Структура

```
├── index.html              # Главная (список заданий)
├── task1/                   # Задание 1 — Структура HTML
├── task2/                   # Задание 2 — Форматирование текста
├── task3/                   # Задание 3 — Списки
├── task4/                   # Задание 4 — Таблица
├── task5/                   # Задание 5 — Изображения и ссылки
├── task6/                   # Задание 6 — CSS-оформление
├── task7/                   # Задание 7 — Мини-сайт (3 страницы) 
│   ├── index.html
│   ├── technologies.html
│   └── resources.html  
└── shared/ 
    ├── style.css 
    ├── script.js
    └── favicon.svg
```

## Технологии

- HTML5
- CSS3 (анимации, Flexbox, Grid, CSS-переменные)
- JavaScript (Intersection Observer, мобильное меню)

## Запуск

```bash
npx -y http-server ./ -p 3003 -c-1 
```

Затем открыть `http://localhost:3003`
